    本工具是用来自动编译的脚本。将下载SDK、下载sdmc patch、下载其他依赖模块、打patch等操作均在脚本中完成，减少因人为因素造成的失误。
1、使用方法
    直接运行 auto_build.sh 脚本，选择对应的项目即可。

2、新建项目
    新建项目只需要在 projects 目录中，新增一个脚本，按照该目录中已有的模块创建即可。

3、sdk 下载的说明
 1)、如果 sdk 是第一次下载，此工具将直接从远程服务器上下载最新的 sdk
 2）、如果 sdk 已经下载，此工具会对比当前的 sdk 与公版 sdk的差异，然后将差异还原，即让 sdk 退回公版的初始状态

4、其他说明
  目前 android P 以下版本不适用
